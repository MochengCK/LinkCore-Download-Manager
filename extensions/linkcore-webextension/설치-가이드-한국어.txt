# LinkCore 브라우저 확장 프로그램 설치 가이드

## 설치 단계

1. **Chrome 브라우저 열기**
   - Chrome 브라우저의 최신 버전을 사용하고 있는지 확인하세요

2. **확장 프로그램 관리 페이지로 이동**
   - 주소창에 `chrome://extensions/` 입력
   - Enter 키를 눌러 확장 프로그램 관리 페이지에 접속

3. **개발자 모드 활성화**
   - 오른쪽 상단 모서리에 있는 "개발자 모드" 토글 찾기
   - 토글을 클릭하여 활성화 (파란색으로 변함)

4. **드래그 앤 드롭 설치**
   - 바탕화면의 `linkcore-webextension` 폴더를 브라우저 창으로 직접 드래그
   - 또는 "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭하고 바탕화면의 `linkcore-webextension` 폴더 선택
   - "폴더 선택"을 클릭하여 설치 완료

**권장 방법**: 폴더를 직접 브라우저 창으로 드래그하여 설치하면 더 빠르고 편리합니다!

## 주의사항

### 보안 경고
설치 중 "이 확장 프로그램은 Chrome 웹 스토어에 등록되어 있지 않습니다"라는 경고가 표시될 수 있습니다. 이는 개발자가 제공하는 확장 프로그램이므로 정상적인 현상입니다.

### 해결 방법
- **방법 1 (권장)**: "설치 계속"을 클릭
- **방법 2 (대체)**: 설치에 실패한 경우 다음 단계를 따르세요:
  1. 주소창에 `chrome://flags/#extensions-on-chrome-urls` 입력
  2. "Extensions on chrome:// URLs"를 Enabled로 설정
  3. 브라우저를 다시 시작한 후 설치 재시도

## 확장 프로그램 기능

설치 후 확장 프로그램은 다음과 같은 기능을 제공합니다:
- 웹 페이지에서 미디어 파일 원클릭 다운로드
- 다양한 파일 형식 지원 (동영상, 오디오, 이미지 등)
- LinkCore 다운로드 관리자와의 원활한 통합
- 다운로드 링크의 스마트 인식

## 자주 묻는 질문

### Q: 설치 후 확장 프로그램이 작동하지 않나요?
A: 다음 항목을 확인해 주세요:
- LinkCore 다운로드 관리자가 실행 중인지 확인
- 브라우저가 재시작되었는지 확인
- 확장 프로그램이 올바르게 활성화되었는지 확인

### Q: 확장 프로그램을 업데이트하려면 어떻게 하나요?
A: 새 버전이 사용 가능해지면 최신 확장 프로그램 폴더를 다시 다운로드하여 설치하세요

### Q: 확장 프로그램을 제거하려면 어떻게 하나요?
A: `chrome://extensions/` 페이지로 이동하여 LinkCore 확장 프로그램을 찾은 후 "제거" 버튼 클릭
